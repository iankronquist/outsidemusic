Quick start
===========

Info on virtualenvs and python:
https://gist.github.com/iankronquist/3b0ac82ec52ed5b77e24

* Make a branch  
```shell
$ git checkout -b my-branch
```
* Make virtualenv  

```shell
$ virtualenv hacking
```

* activate virtualenv  

```shell
$ source hacking/bin/activate
```

* install deps  

```shell
$ pip install -r requirements.txt
```

* go hack
